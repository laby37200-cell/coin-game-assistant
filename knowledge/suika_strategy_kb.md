# 수박게임 전문 공략 지식 베이스

## 핵심 전략 요약

### 1. 기본 배치 원칙

#### 좌우 정렬 전략
- **큰 과일(동전)은 왼쪽**: 멜론, 파인애플 등 큰 과일은 항상 왼쪽 구역에 배치
- **작은 과일은 오른쪽**: 체리, 딸기 등 작은 과일은 오른쪽에서 처리
- **중앙은 유동 공간**: 연쇄 합체를 위한 버퍼 공간

#### 높이 관리
- **고도 차이 2단계 이하 유지**: 타워 형성 방지
- **경사면 생성**: 굴림 제어를 위한 자연스러운 경사
- **30% 여유 공간**: 최소 30% 이상의 여유 공간 항시 유지

### 2. 연쇄 합체 기술

#### 황금 연쇄 패턴
```
작은 과일 + 작은 과일 → 중간 과일
중간 과일 + 중간 과일 → 큰 과일
큰 과일 + 큰 과일 → 최대 과일
= 3초 만에 500점 이상
```

#### 준비 요구사항
1. **동일 과일 사전 배치**: 합체할 과일을 미리 인접하게 배치
2. **물리 엔진 활용**: 굴림과 바운스를 이용해 자연스럽게 합체 유도
3. **타이밍 완벽**: 0.5초 이내 최적 선택

### 3. 물리 엔진 제어

#### 굴림 역학 공식
```
착지 위치 = 기준 중심 + (낙하 높이 × 0.1) × 방향
```

**실제 적용**:
- 오른쪽 굴림: 중심 약간 오른쪽 착지
- 왼쪽 굴림: 중심 약간 왼쪽 착지
- 수직 쌓기: 정중앙 적중
- 속도 제어: 높이 조절

#### 바운스 제어 매트릭스

| 낙하 높이 | 바운스 수준 | 최적 사용 사례 |
|-----------|-------------|----------------|
| 1-2단계   | 최소        | 정밀 쌓기      |
| 3-4단계   | 중간        | 부드러운 재배치 |
| 5단계+    | 높음        | 비상 간격 조정 |

### 4. 고급 배치 이론

#### 6구역 관리 시스템
```
[A1][A2] - 큰 과일 전용 (왼쪽)
[B1][B2] - 중간 과일 전환 (중앙)
[C1][C2] - 작은 과일 처리 (오른쪽)
```

각 구역은 특정 역할과 규칙 보유:
- **A구역**: 최종 목표 과일 준비 공간
- **B구역**: 연쇄 합체 실행 공간
- **C구역**: 작은 과일 즉시 처리

### 5. 점수대별 전략

#### 0-1000점: 기초 다지기
- **목표**: 안정적인 배치 구조 확립
- **핵심**: 기본 분리 전략 마스터
- **함정**: 서두르지 말 것

#### 1000-2000점: 중반 최적화
- **목표**: 효율적인 공간 활용
- **핵심**: 3수 이상 앞 계획
- **함정**: 작은 과일 3개 이상 축적 금지

#### 2000-3000점: 연쇄 준비
- **목표**: 연쇄 합체 최대화
- **핵심**: 동일 과일 사전 배치
- **함정**: 욕심 부리지 말 것

#### 3000-5000점: 엘리트 영역
- **목표**: 물리 엔진 완벽 이해
- **핵심**: 5수 이상 앞 읽기
- **함정**: 긴장하지 말 것

#### 5000점 이상: 프로페셔널
- **목표**: 픽셀 완벽 정밀도
- **핵심**: 직관과 계산의 조화
- **함정**: 자만하지 말 것

### 6. 피해야 할 나쁜 습관

1. **패닉 드롭**: 천천히, 먼저 생각
2. **편향된 배치**: 전체 컨테이너 폭 사용
3. **작은 과일 무시**: 성공을 결정짓는 요소
4. **운 의존**: 일관된 전략 생성
5. **조기 포기**: 게임 오버까지 싸우기

### 7. 고급 기술 모음

#### 벽 타기
- 컨테이너 가장자리 제어 활용
- 벽을 이용한 정밀한 위치 조정

#### 과일 샌드위치
- 지연 합체를 위한 과일 가두기
- 타이밍 제어를 위한 버퍼 생성

#### 캐스케이드 엔지니어링
- 5+ 연쇄 반응 구축
- 계획된 연쇄로 최대 점수 획득

#### 비상 피벗
- 게임 중 완전한 전략 변경
- 위기 상황에서 즉각 대응

## 동전게임 적용 원칙

### 1. 동전 크기별 배치
- **레벨 1-3 (작은 동전)**: 오른쪽 구역에서 즉시 합체
- **레벨 4-6 (중간 동전)**: 중앙 구역에서 연쇄 준비
- **레벨 7-11 (큰 동전)**: 왼쪽 구역에 고정 배치

### 2. 랜덤 드롭 대응
- 랜덤 동전은 레벨 1-6까지만 가능
- 작은 동전이 떨어질 때 즉시 처리 공간 확보
- 큰 동전 주변에 작은 동전이 끼지 않도록 주의

### 3. 물리 파라미터 최적화
- **중력**: 낙하 속도 제어 (기본값: 900)
- **마찰**: 굴림 제어 (기본값: 0.5)
- **탄성**: 바운스 제어 (기본값: 0.3)

### 4. 평가 함수 가중치
```python
Score = 
  + (큰 동전 개수 × 10)        # 큰 동전 보너스
  + (인접도 × 5)                # 같은 동전 인접도
  + (구석 배치 × 3)             # 구석 배치 보너스
  - (높이 × 2)                  # 높이 페널티
  - (블로킹 × 20)               # 블로킹 페널티
  + (좌우 정렬 보너스)          # 정렬 보너스
```
